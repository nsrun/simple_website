## Simple Portfolio Website 🌟

## Overview
A modern, responsive portfolio website built with vanilla HTML5, CSS3 (with CSS custom properties and dark mode), and JavaScript. Features smooth animations, theme toggle, contact form, and scroll effects.

![Screenshot](./screenshot.png) <!-- Add a screenshot here -->

## Features
- ✅ **Dark/Light Theme Toggle** (persists in localStorage)
- ✅ **Smooth Scrolling** for anchor links
- ✅ **Intersection Observer Animations** (fade-in on scroll)
- ✅ **Interactive Contact Form** (with validation)
- ✅ **Hover Effects** on service cards
- ✅ **Fully Responsive** (mobile-first design)
- ✅ **Modern CSS** (Grid, Flexbox, Custom Properties)
- ✅ **Performance Optimized** (no frameworks, ~5KB total)

## Tech Stack
```
HTML5 | CSS3 | Vanilla JavaScript | IntersectionObserver API
Google Fonts (Inter) | CSS Grid & Flexbox
```

## Sections
1. **Hero** - Introduction with CTA buttons
2. **About** - Bio and skills showcase
3. **Services** - Web Design, Frontend, Backend
4. **Contact** - Working form with JS validation

## Quick Start 🚀
1. Open `index.html` in any modern browser:
   ```
   # Windows
   start index.html
   
   # macOS
   open index.html
   
   # Or just double-click!
   ```
2. Toggle theme (top-right button)
3. Test form submission
4. Resize browser for responsive design

## Customization
- **Colors**: Edit CSS custom properties in `:root`
- **Content**: Update text in `index.html`
- **Skills/Services**: Modify `.skills-grid` and `.services-grid`
- **Theme**: Toggle `data-theme="dark"` on `<body>`

## Scripts
- `script-clean.js`: Production JS (loaded in index.html)
- `script.js`: Backup with @ts-nocheck for VSCode

- **Live Reload**: Use VSCode Live Server extension
